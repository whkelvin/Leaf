# Leaf
Leaf 是一款支持直式閱讀的免費電子書閱讀器。

### 需求

*目前僅支持Epub檔案

*支持Windows 7以上版本

*IOS, Android, Mac 版本正在開發中

### 設置
下載並執行LeafReaderSetup.msi
